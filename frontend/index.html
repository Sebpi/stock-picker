Setup Guide

1
Add stocks to your Watchlist

Only watchlist stocks are monitored. Use the Screener tab to find and add stocks.

2
Configure Email (Gmail)

In your .env file set SMTP_USER, SMTP_PASS (Gmail App Password), and ALERT_EMAIL. Get an App Password at myaccount.google.com/apppasswords.

3
Configure SMS via Twilio (optional)

Sign up free at twilio.com, then set TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, TWILIO_FROM_NUMBER, TWILIO_TO_NUMBER in your .env file.

4
Restart the backend

After editing .env, restart uvicorn. Then click Send Test Alert to verify.
